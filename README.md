Trabalho final Front-End
Para finalização do curso de Front-end estarei desenvolvendo um site para venda de moda circular. Nos últimos tempos a moda circular tem tipo bastante procura, pois muitas roupas acabam não sendo usadas são compradas por impulso, outras estão em tão bom estado que podem ser úteis a outras pessoas. Além disso tem aquelas pessoas que buscam roupas que marcam pela autenticidade ou seja peças exclusivas. 
Para execução deste projeto estarei utilizando HTML5 para organização semântica do conteúdo. Na parte de estilização do site usarei CSS3 com layout responsivo. A interatividade e lógica ficará por conta das técnicas do JavaScript.
